# Roomify / BAT - POC

Roomify Booking and Availability Management Library proof of concept.
See: https://github.com/Roomify/bat

### Usage

```
git clone git@github.com:ircykk/roomify-bat-poc.git
cd roomify-bat-poc
composer install
php poc.php
```
